Game universe - game help (only english)

Game start screen:

File menu:

  1. Save result -> saves ended game result to text file.
  2. Print result -> prints ended game result.
  3. Exit -> exit a program.

Game menu:

  1. New -> New game
  2. Load -> load saved game
  3. Save -> save current game
  4. Undo last move
  5. Redo last undoed move
  6. Replay game. Possible current game and saved games.
  7. Settings -> Open game settings

Settings screen:

Game settings to change:

  1. Basic properties
    1. Gametype: Checker / Breakthrough
    2. Board size: 8X8 to 16X16
    3. Language: english (ENG) or finnish (FIN)
  2. Board properties
    1. Board grid colors.
  3. Player properties:
    1. Player symbol color
    2. Player symbol (ellipse, triangle or cubic)

Help menu:

  1. View help -> this help view
  2. About -> About this game.

New game

Breakthrough game

Game rules:

  1. Player can only go vertical (3 possibles)
  2. Player cannot go vertical direction if other player mark is front of it.
  3. Game winner is player which manage get own mark to opposite side.

Checker game

Game rules:

  1. Player can only move mark two possible way. Left or right corner.
  2. If player has change to eat opposite player mark its forced to do so.
  3. If player manage get to opposite side, mark goes "king".
  4. King mark can go both ways four possible direction.
  5. Winner is player which eat all opposite player marks.
